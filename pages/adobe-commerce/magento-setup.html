Local Magento Setup

A comprehensive guide to setting up Magento 2 on your local machine. This guide covers the complete installation process from prerequisites to post-installation configuration.

System Requirements

Ensure your system meets these minimum requirements:

  • Operating System - macOS, Linux, or Windows with WSL2
  • PHP - 8.1 or 8.2 (Apache or Nginx)
  • MySQL - 8.0 or MariaDB 10.4+
  • RAM - Minimum 2GB, recommended 4GB+
  • Disk Space - At least 5GB for full installation
  • Composer - Latest version

Installation Guide

Follow this step-by-step process to install Magento locally:

Step 1: Prepare Your Environment

Install required packages and tools:

# For macOS with Homebrew brew install php@8.2 composer mysql # For Ubuntu/Debian sudo apt-get install php8.2 composer mysql-server

Step 2: Download Magento

Download Magento 2 using Composer:

composer create-project --repository-url=https://repo.magento.com/ magento/project-community-edition magento2-local

Step 3: Configure Database

Create a database and user for Magento:

mysql -u root -p CREATE DATABASE magento2; CREATE USER 'magento'@'localhost' IDENTIFIED BY 'password'; GRANT ALL PRIVILEGES ON magento2.* TO 'magento'@'localhost'; FLUSH PRIVILEGES;

Step 4: Install Magento

Run the Magento installation command:

cd magento2-local php bin/magento setup:install \ --base-url=http://localhost/magento2-local/ \ --db-host=localhost \ --db-name=magento2 \ --db-user=magento \ --db-password=password \ --admin-firstname=Admin \ --admin-lastname=User \ --admin-email=admin@example.com \ --admin-user=admin \ --admin-password=Admin123!@# \ --language=en_US \ --currency=USD \ --timezone=America/Chicago

Step 5: Verify Installation

Check that Magento is installed correctly:

php bin/magento setup:di:compile php bin/magento setup:static-content:deploy -f php bin/magento cache:clean

Post-Installation Configuration

After installation, configure these important settings:

Web Server Configuration

Set up virtual host for your Magento installation. For Apache, create a virtual host file:

<VirtualHost *:80> ServerName magento2.local DocumentRoot /path/to/magento2-local/pub <Directory /path/to/magento2-local/pub> AllowOverride All Require all granted </Directory> </VirtualHost>

File Permissions

Set proper file permissions for Magento directories:

cd /path/to/magento2-local find . -type d -exec chmod 755 {} \; find . -type f -exec chmod 644 {} \; chmod u+x bin/magento

Development Mode

Switch to developer mode for better debugging:

php bin/magento deploy:mode:set developer

In developer mode, static files are created on-demand and slower performance is expected. This mode is ideal for development.

Essential Magento Commands

  • php bin/magento cache:clean - Clear all caches
  • php bin/magento cache:flush - Flush all caches
  • php bin/magento indexer:reindex - Reindex all data
  • php bin/magento setup:upgrade - Run database upgrades
  • php bin/magento admin:user:create - Create admin user
