Agent Driven Component Development

@agent-generator analyzes your AEM codebase to automatically generate specialized project-specific agents customized to your exact tech stack, patterns, and workflows. Instead of using generic templates, it reads your Java services, React components, build configurations, and naming conventions to create agents that reference your actual code patterns and tools.

The generated agents include core orchestrators (@conductor, @builder), specialist experts for backend/frontend/integrations/quality, and knowledge keepers for architecture and standards. Each agent is 200-500 lines of specific, production-ready content with real examples from YOUR codebase, complete cross-references, and ready-to-use immediately after generation.
